# AQA GCSE 8700 — AO2

> **AO2** — Explain, comment on and analyse how writers use language and structure to achieve effects and influence readers, using relevant subject terminology to support their views.

*(Verbatim spec text private side only — © AQA)*

## Tests canonical

- [[../../../02-curriculum/ks4/english/language-and-structure-analysis|KS4 English — Language and structure analysis]]

## Where assessed

Both papers, Section A. The "use of subject terminology" requirement is explicit in AQA's mark scheme — students need to name techniques accurately (alliteration, sibilance, anaphora, hypophora, etc.).
